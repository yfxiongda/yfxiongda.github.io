Choose font

First of all, you need to choose a font. I prefer Noto Color Emoji, which is a font used to display emojis on Android. <?xml version="1.0" encoding="UTF-8"?>

    <family>sans-serif</family>

  </alias>
  <alias>
    <family>monospace</family>
    <prefer>
      <family>Noto Color Emoji</family>
    </prefer>
  </alias>
</fontconfig>
